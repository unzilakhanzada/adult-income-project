# Adult Income Prediction Project

## 📊 Project Overview

This project predicts whether a person's income is greater than 50K per year using machine learning techniques. The model is trained on the Adult Income dataset.

## 📂 Dataset

Dataset used: Adult Income Dataset
This dataset contains demographic information such as age, education, occupation, work class, etc.

## 🐍 Programming Language

Python

## 📚 Libraries Used

* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Joblib

## 🤖 Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest
* Voting Classifier (Ensemble Model)

## ⚙️ Data Preprocessing

* Handling Missing Values (SimpleImputer)
* One Hot Encoding
* Feature Scaling (StandardScaler)
* Train-Test Split

## 🔎 Hyperparameter Tuning

* GridSearchCV
* RandomizedSearchCV

## 📈 Model Evaluation

* Accuracy Score
* Confusion Matrix
* Classification Report

## 💾 Model Saving

The trained model is saved using Joblib for future predictions.

## 📁 Project Files

* adult.csv → dataset
* adult_income.ipynb → notebook containing complete code
* model.pkl → trained machine learning model

## 🎯 Goal

To build an accurate machine learning model that can classify whether a person's income is above or below 50K.
